A website for a non-profit, christian organization.

The Problem

A Couple with a call from the lord; To raise funds and help struggling churches, want a way to reach the whole world. A website was the right solution. I had to design and build a website that will be a central portal for this initiative.

Industry

  • Christianity, Charity

Solutions

  • Web design
  • Web development

Country

  • Eswatini

Tech Stack

  • Next Js
  • Tailwind CSS
  • Vercel

The Solution

I built a clean, fast website using Next.js and Tailwind CSS, and hosted it on Vercel. The site highlights the organization’s mission and includes simple forms for donations and church support requests. It’s easy to use on both desktop and mobile, helping the organization reach more people and raise funds online.
